A lightweight, interactive web application that leverages Google's advanced Gemini Large Language Models (LLMs) to analyze news articles and determine their authenticity. This tool acts as a professional fact-checker, identifying misinformation, bias, and logical fallacies in real-time.
✨ Features
Real-Time Analysis: Instant evaluation of pasted news text.

Advanced AI Reasoning: Powered by Google's Gemini Flash models for high-speed, accurate contextual understanding.

Detailed Verdicts: Categorizes news as REAL, FAKE, or MISLEADING, complete with a bulleted breakdown of the AI's reasoning.

User-Friendly Interface: Built with Streamlit for a clean, responsive web UI.
